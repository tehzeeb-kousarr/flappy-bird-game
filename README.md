# 🐦 Flappy Bird - Unity Game

A simple and addictive **Flappy Bird clone** developed using the **Unity Engine**. The game challenges players to navigate a bird through pipes by timing jumps perfectly without crashing.

## 🚀 Installation & Setup

Follow these steps to run the project on your system:

1. **Download or Clone the Repository**

   ```bash
   git clone https://github.com/your-username/flappy-bird-unity.git
   ```

2. **Open in Unity**

   * Open Unity Hub
   * Click on **Add Project**
   * Select the cloned folder

3. **Load the Game Scene**

   * Navigate to `Assets/Scenes/`
   * Open the main scene (e.g., `GameScene.unity`)

4. **Run the Game**

   * Press the **Play Button ▶️** in Unity Editor

<br>

## 🎯 How to Play

* Tap / Click / Spacebar to make the bird jump
* Avoid hitting pipes or the ground
* Survive as long as possible to increase your score 🏆

<br>
